plantuml-skill

通过 Kroki 渲染 API 支持 10+ 种图表 —— 无需 Java、无需 Graphviz、无需本地安装。本地 Docker 和 plantuml.jar 备选随包附带。

/plugin marketplace add Agents365-ai/365-skills
GitHub ClawHub SkillsMP Discord

由一句自然语言提示词生成 — 完整提示见 README。

为什么选这个技能

LLM 本身懂 PlantUML 语法。本技能补齐渲染管道、常见错误防护、以及三种后端方案 —— 想在哪渲染都行。

🧩

10+ 种图表类型

时序图、组件图、类图、ER 图、活动图、用例图、状态图、C4、思维导图、Gantt —— 每种都有地道的语法模板与形状词汇。

🌐

三种渲染后端

公共 Kroki API(零安装)、本地 Kroki Docker(离线)、plantuml.jar + Java(气隙环境)—— 按需选用。

🎨

内置主题

plain、cerulean、blueprint、aws-orange、vibrant —— 加完整的 skinparam 覆盖自定义。

🔗

C4 真的能用

PlantUML 公共服务器的 !include 经常 404。技能使用 Kroki 的 c4plantuml 端点并预合并 includes —— 默认就稳。

⚠️

常见错误防护

精选 9 行 Common Mistakes:箭头方向、布局溢出、标签转义、参与者顺序、C4 includes 陷阱等等。

🚀

零安装默认值

curl 在 macOS、Linux、Windows(Git Bash / WSL)上预装,不需要 Node、不需要 Java、不需要配置。

安装

选择你的平台,只要 10 秒。

# Plugin marketplace (recommended)
/plugin marketplace add Agents365-ai/365-skills
/plugin install plantuml

# Or any agent
npx skills add Agents365-ai/365-skills -g

# Manual install
git clone https://github.com/Agents365-ai/plantuml-skill.git ~/.claude/skills/plantuml-skill
# Global
git clone https://github.com/Agents365-ai/plantuml-skill.git ~/.config/opencode/skills/plantuml-skill

# Project
git clone https://github.com/Agents365-ai/plantuml-skill.git .opencode/skills/plantuml-skill
# Global
git clone https://github.com/Agents365-ai/plantuml-skill.git ~/.openclaw/skills/plantuml-skill
# Via SkillsMP CLI
skills install plantuml-skill
# User-level
